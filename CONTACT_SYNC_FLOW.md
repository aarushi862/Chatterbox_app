## 🔐 Backend API Flow

```
┌─────────────────────────────────────────────────────────────────┐
│          POST /api/contacts/sync                                 │
│          Headers: { Authorization: "Bearer <token>" }            │
│          Body: { emails: [...], phones: [...] }                  │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                  AUTHENTICATION                                  │
│  authMiddleware.protect() → Verify JWT token                    │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                  INPUT VALIDATION                                │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  • Check emails and phones arrays exist                  │   │
│  │  • Limit to 100 contacts                                 │   │
│  │  • Filter empty values                                   │   │
│  └──────────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                  NORMALIZATION                                   │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  Emails:                                                 │   │
│  │  • Convert to lowercase                                  │   │
│  │  • Trim whitespace                                       │   │
│  │                                                           │   │
│  │  Phones:                                                 │   │
│  │  • Remove all non-digits                                 │   │
│  │  • Filter length >= 10                                   │   │
│  └──────────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                  DATABASE QUERY                                  │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  User.find({                                             │   │
│  │    _id: { $ne: req.user._id },  // Exclude self         │   │
│  │    $or: [                                                │   │
│  │      { email: { $in: normalizedEmails } },              │   │
│  │      { phone: { $in: normalizedPhones } }               │   │
│  │    ]                                                     │   │
│  │  }).select('name email phone avatar')                   │   │
│  └──────────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                  RESULT PROCESSING                               │
│  ┌──────────────────────────────────────────────────────────┐   │
│  │  1. Extract matched emails and phones                    │   │
│  │  2. Build registered array with user objects             │   │
│  │  3. Build unregistered array with remaining contacts     │   │
│  │  4. Calculate stats (total, found, notFound)             │   │
│  └──────────────────────────────────────────────────────────┘   │
└────────────────────────┬────────────────────────────────────────┘
                         │
                         ▼
┌─────────────────────────────────────────────────────────────────┐
│                  RESPONSE                                        │
│  {                                                               │
│    registered: [{ user: {...}, matchedBy: "..." }],             │
│    unregistered: [{ email: "...", phone: "...", name: "..." }], │
│    stats: { total: 10, found: 3, notFound: 7 }                  │
│  }                                                               │
└─────────────────────────────────────────────────────────────────┘
```

## 🎨 UI State Machine

```
┌─────────────┐
│  INITIAL    │
│  (closed)   │
└──────┬──────┘
       │ User clicks Sync button
       ▼
┌─────────────┐
│ PERMISSION  │◄──────────┐
│  (step 1)   │           │
└──────┬──────┘           │
       │                  │
       │ User clicks      │ Error or
       │ Sync/Manual      │ Permission denied
       ▼                  │
┌─────────────┐           │
│  SYNCING    │───────────┘
│  (step 2)   │
└──────┬──────┘
       │ API success
       ▼
┌─────────────┐
│  RESULTS    │
│  (step 3)   │
└──────┬──────┘
       │ User clicks Done/Close
       ▼
┌─────────────┐
│   CLOSED    │
│ (badge      │
│  updated)   │
└─────────────┘
```
